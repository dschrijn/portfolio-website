HARMONIOUS

A meditation app that focuses on mindfulness and meditation to relieve mental health issues like stress, anxiety, and depression.

About the Project

The Challenge

How might we introduce meditation to new users that offers deeper and insightful content in a simple way that promotes engagement and knowledge retention while providing a clear understanding of the value and overall user experience.

My Role

I conducted user research. I produced sketches, wireframes and mockups. I created a Design Language System and conducted multiple rounds of testing.

The Process

Define

Understanding the Problem

The goal is to design an intuitive, user friendly, and stylish meditation application that targets new and existing users that encourages them to practice and live a healthier life through mindfulness. By conducting a Competitive Analysis, I defined the following problem statement.

Problem Statement

“Our users need a way to easily identify their specific stress and receive content that is intuitive & engaging because they would like a safe place to practice meditation so they can reduce their mental pains.” We will know this to be true when we see how users are spending more time meditating through their tailored and guided lesson plan, and in turn reap the many health benefits of meditating.

Empathize & Ideate

User Research

By conducting user interviews, I gained insight on the users lifestyle, interests/hobbies, challenges, and goals. Based on the findings, I created user personas, journey maps, and user flows.

Prototype

Wireframes

Based on the user research, I started with sketches and moved on to low-fidelity wireframes to get a basic layout and structure. Once the basic functionality was defined, I designed High-Fidelity mockups focusing on Color, Typography, Iconography, and Imagery which I have detailed in the Design Language System.

Test

I conducted six remote moderated usability tests. The goal of this study was to assess the learnability, efficiency, and satisfaction for new users interacting with the app for the first time on a mobile device.

Report Findings

Based on the results of the usability tests, I created a Rainbow Spreadsheet to organize the information and create visual patterns. I was able to identify 5 key issues. For this iteration, each issue has been updated based on the reported evidence.

Design Language System

In order to stay consistent with my own design, I created a Design Language System. This would ensure no matter what platform I design on, there would be a cohesive user experience.
