From Pixels to Animation

The process in taking an illustration and bringing it to life with a little code and some animation fundamentals.

Role:

  • Front-End Developer

Tools:

  • Sketch App
  • Figma
  • Atom
  • Chrome Inspector

Thumbnails

I started by sketching out how I wanted each element to be animated.

SVG Preparation & Export

Once the thumbnails were completed, I used Figma to configure the SVG file to group each element that I wanted to animate.

Animation

Ship

I started by animating the ship to hover by simply moving up and down.

Light Beams

Next I animated the light beams to animate up and start again from the bottom in an infinite loop, making it look like it was lifting the alien.

This seemed really dull and unrealistic. I wanted the beams of light to randomly start at a certain point and time, so I implemented a function/method that would achieve just that.

Alien 1

For Alien 1, I wanted the alien to seem scared and confused. I started with both arms waving back and forth, then had each leg kicking in and out. Next I added a basic facial animation with the mouth scaling and the pupil translating left to right. Lastly, I animated the body to move up and down and the hat to do the same but with a slight rotation.

Alien 2

For Alien 2, I wanted the alien to seem frantic and point towards the ship. I started with animating the legs up and down. Next, I added a scale to mimic the body shaking. I positioned both arms in place and applied a slight rotation that syncâ€™d with the timing of the body. Lastly, the pupil has a slight translate movement and I applied a scale to the mouth to increase and shrink.

Results

By applying CSS and the basic principles of animation, I was able to take an illustration and create a fun animation. While working on this project, I learned how CSS and Keyframes can configure an elements style. Keyframes have many properties that allow creative interactions, which I only scratched the surface using transitions. Even though you can use only CSS Animation to achieve your goal, knowing JavaScript is extremely helpful for complex animations.
